# To-Do-App
A simple Java console-based To-Do List application that allows users to manage daily tasks. 
Users can add tasks, view tasks, mark tasks as completed, and delete tasks using a menu-driven interface.
##  Features
-  Add new tasks  
-  View all tasks  
-  Mark tasks as completed  
-  Delete tasks  
-  Exit the application

   ## Project Structure

##  Technologies Used
- Java
- ArrayList
- Scanner
- Console-based UI 
##  How to Run the Application
### Clone the Repository
### Navigate to the Project Folder
### Compile the Java Files
javac ToDOApp.java Task.java
### Run the Application
java ToDOApp
